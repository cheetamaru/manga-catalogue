## Install env and dependencies:
```
pip install virtualenv

virtualenv env

pip install -r requirements.txt
```

## Migrate models:
```
env\scripts\activate

python manage.py migrate
```

## Run server:
```
env\scripts\activate

python manage.py runserver
```
