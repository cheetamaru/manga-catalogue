## Install env:
```
pip install virtualenv

virtualenv env
```

## Activate:

Windows
```
env\scripts\activate
```
Mac
```
source env/bin/activate
```

## Install requirements:
```
pip install -r requirements.txt
```

## Migrate models:
```
env\scripts\activate

python manage.py migrate
```

## Run server:
```
env\scripts\activate

python manage.py runserver
```
